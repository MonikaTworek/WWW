iframe ≡ inline frame (ramka wbudowana)
Konstrukcja 
<iframe id = "myIFrame" src="http://cs.pwr.edu.pl/cichon/2017_18_b/WWW/W6/test04.php">
Styl 
#myIFrame{
  width: 100%;
  margin: 0;
  border: 2px solid #555;
  height: 15em;
  frameborder  : 0;
  scrolling    : auto;
}
